# Macro analysis: ECB key rate vs Eurozone inflation (2015-2025) 
Analysis of the relationship between ECB monetary policy and inflation dynamics in the EuroZone, using Python and FRED macro economics data  

## Data sources: 
- ECB deposit facility rate (ECBDFR), FRED
- Eurozone HICP inflation (CP0000EZ19M086NEST), FRED

## Tools: 
Python, Pandas, NumPy, Matplotlib, Google colab

## Key finding:
correlation between ECB key rate and eurozone inflation: 0.25. The low correlation reflects the stuctural transmission lag of monetary policy.
The ECB reacts to observed inflation, not in anticipation. The 2022-2024 tightening cycles clearly illustrates this delay.
